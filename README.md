# 📊 Grafici con Python

Questo progetto genera diversi tipi di grafici usando **Matplotlib** e **Seaborn**.

## 📌 Funzionalità
- 📈 **Grafico a linee** con dati casuali.
- 📊 **Istogramma** basato su distribuzione normale.
- 🖼️ **Salvataggio delle immagini** in formato PNG.
